# gingerol-doe

漢方中の **10-gingerol** を HPLC で頑健に分離・定量するための、条件最適化プロジェクト。
カラム温度 T・流速 F・溶媒比 φ（水:ACN）の3条件を統計的に最適化し、**デザインスペース**（ブレに強い条件の領域）を3Dで可視化する。最終的に手順を再利用可能な skill に切り出す。

> 状態：**足場のみ（空の器）**。スクリプト本体は設計を固めてから実装する。

## 背景（なぜ）
現行条件（T=75℃ / F=0.6 mL/min / 水:ACN=60:40 アイソクラティック）では、目的ピークが前後2つの夾雑ピークに挟まれており、条件のブレで分離度が落ちるリスクがある。条件を変えると夾雑ピークと溶出順が入れ替わる。

## 設計の考え方（確定済みの要点）
- **流速 F は保持係数 k を変えない。** 保持の化学は T と φ だけで決まる。
  - 保持：`k(T, φ)` を3ピーク分フィット。保持時間は `t_R = (V_m/F)(1+k)`、`t_0 = V_m/F` で復元（F は既知の式で入る）。
  - 幅：`W(T, φ, F)` を3ピーク分（F が主役）。
- 保持の心臓部は対数変換で重回帰になる：`ln k = a + b/T[K] + c·φ`（必要なら +d·φ²）。
  下敷きはファントホッフ式（温度）・LSS/Snyder（溶媒）・van Deemter（幅）。
- **V_m はカラム固有の定数**。保持されない物質を1回注入して `V_m=t_0×F` で一度だけ確定。
