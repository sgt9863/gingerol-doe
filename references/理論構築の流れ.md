## 0. 結論を1枚で

漢方中の目的成分 **10-gingerol（TP）** を、前後に溶出する夾雑成分（IP1, IP2）から
**頑健に分離する HPLC 条件**を、当てずっぽうでなく「物理化学のメカニズム＋統計（実験計画法）」で
求める。具体的には、カラム温度 T・移動相の有機溶媒比 φ・流速 F の3つを動かし、
**分離度 Rs が十分高く（≥2.0）、かつ条件が多少ブレても合格に留まる領域（デザインスペース）**を
3D で可視化し、その中心（最も余裕のある点）を推奨条件とする。

```
問題設定 → メカニズムモデル → 実験計画(DoE) → 実測フィット → デザインスペース → 推奨条件
  §1          §3                §5             §4(統計)     §6           §6
```

## 1. 解こうとしている問題

- 目的成分 TP（10-gingerol）が、前後の夾雑ピーク IP1・IP2 に挟まれている。
- 条件（温度・溶媒比・流速）が少し変わると、ピークが重なって分離できなくなるリスクがある。
- さらに**クロスオーバー**（条件次第で溶出の順番が入れ替わる）が起こりうる。
- gingerol 類を逆相カラム・アセトニトリル(ACN)系で分けるのは分析化学的に定番であり、
  課題設定そのものは現実的[文献15, 16]。

**ゴール**：3条件の空間の中で「いつも well-separated」な領域を、根拠をもって特定する。

## 2. なぜ「ただの2次回帰」でなく「メカニズムモデル」か

実験データに対して `y = a + bT + cφ + …` という汎用の多項式を当てることもできる。
しかし本プロジェクトは**クロマトグラフィーの物理化学**に基づく式を使う。理由：

- 少ない実験点でも、**外挿や条件間の挙動が物理的に妥当**になりやすい。
- パラメータが「温度の効き」「溶媒の効き」など**意味を持つ**ので解釈・再利用できる。
- クロスオーバーのような現象を、交互作用項として**自然に表現**できる。

> **統計メモ（モデルの型）**：統計検定2級でいう「回帰モデル」は説明変数 x から応答 y を
> 予測する式。ここでは y を「保持の強さ（後述の ln k）」、x を「温度・溶媒比の関数」にとる。
> ポイントは、後述のとおりこの式が**パラメータについては線形**（足し算）である点。

## 3. メカニズムモデル（コードの土台）

### 3.1 保持係数 k —「どれだけ長くカラムに留まるか」

- **保持係数 k**：成分がカラム内の固定相にどれだけ強く引き留められるかを表す無次元の量。
  k が大きいほど遅く出てくる。
- 保持時間 t_R（ピークが出る時刻）とは `t_R = t_0·(1 + k)` の関係。
  - `t_0`（ボイド時間）＝保持されない物質が通り抜ける時間 ＝ `V_m / F`。
  - `V_m`＝カラム内で移動相（液）が占める体積（死容量）。`F`＝流速。

### 3.2 温度の効き — ファントホッフ式

分配平衡の熱力学から、保持の対数は絶対温度の逆数に比例する：

```
ln k = a + b/T_K       （T_K = 温度[℃] + 273.15、単位はケルビン K）
  a : 切片
  b : 温度項の係数（b>0 なら低温ほど保持が強い）
```

これは「平衡定数の対数が 1/T に比例する」というファントホッフの関係をクロマトに適用したもの[文献8]。
実際には曲がる（非線形になる）こともあるが、逆相で疎水性相互作用が支配的なら直線で近似でき、
必要なら高次項で補正してよい[文献2]。

> **統計メモ（変数変換）**：「ln k を 1/T_K に対して直線で表す」は、統計2級の単回帰と同じ発想。
> 縦軸に ln k、横軸に 1/T_K をとると直線になる、という変数変換である。

### 3.3 溶媒の効き — LSS（線形溶媒強度）モデル

逆相 HPLC では、有機溶媒(ACN)の体積分率 φ を増やすほど成分は速く出る。広い範囲で
**log k が φ の一次式**になることが知られる（Linear Solvent Strength）[文献4, 8]：

```
ln k = ln k_w − S·φ      （φ = ACN 体積分率, 0〜1）
  ln k_w : φ=0（純水）での保持
  S      : 溶媒強度の傾き（大きいほど φ に敏感）
```

本プロジェクトはこれを温度項と足し合わせる（係数 c が −S に相当）：

```
ln k = a + b/T_K + c·φ (+ d·φ²)
  c : 溶媒1次項（c<0 なら ACN 増で保持が下がる）
  d : 溶媒2次項（曲がりの補正。不要なら 0）
```

### 3.4 交互作用とクロスオーバー

温度と溶媒の効きは独立でなく、**溶媒の傾き S が温度で変わる**。これを交互作用項で表す：

```
ln k = a + b/T_K + c·φ + d·φ² + e·(φ/T_K)
  e : 温度×溶媒の交互作用項
```

この `e·φ/T_K` が**ピークごとに違う**と、ある条件では IP1<TP<IP2、別の条件では順番が
入れ替わる、という**クロスオーバー**が生じる。後述の分離度はこれに対応する。

> **統計メモ（交互作用）**：2級でいう「交互作用」＝ある要因の効果が別の要因の水準で変わること。
> ここでは温度の水準によって「溶媒を増やしたときの保持低下の度合い」が変わる、という意味。

### 3.5 保持時間の復元

ln k から k を戻し（`k = e^{ln k}`）、保持時間にする：

```
t_R = (V_m / F)·(1 + k)
```

ここで重要：**この変換に指数 e^{ } が入るため、t_R は係数 a,b,c,… について「非線形」**になる
（§4.3 で効いてくる）。一方、ln k 自体は係数について線形のまま。

### 3.6 ピーク幅とカラム効率 — van Deemter 式

ピークの鋭さ（幅）は古典的には流速で決まる。段高 H（小さいほど高効率）を3項の和で表す[文献6, 7]：

```
H = A + B/u + C·u        （u = 線速度 = L·F / V_m、L=カラム長）
  A : 渦拡散（充填の乱れ）
  B/u : 縦拡散（遅いほど広がる）
  C·u : 物質移動抵抗（速いほど広がる）
```

**ただし本プロジェクトでは温度項・溶媒項を加えて拡張する**。理由：u は流速 F でしか動かず、
DoE の F 範囲が狭い（例 0.55〜0.65 mL/min）ため u はほとんど変化しない。一方、実測では段高 H が
温度 T と溶媒比 φ で大きく動く（k の変化に伴う物質移動・拡散の変化）。そこで：

```
H = A + B/u + C·u + D·φ + E/T_K
  D : 溶媒項（φ が段高に与える効果）
  E : 温度項（1/T_K が段高に与える効果）
```

実データで「u のみ」から「u＋φ＋1/T_K」にすると幅の R² が改善した（例：TP 0.03→0.42、
IP1 0.27→0.71、IP2 0.17→0.38）。ただし幅は分散現象でノイズが大きく、保持（R²≈1.0）ほどは
上がらない。点数が少ない場合は過学習を避け、φ項・T項は自由度が確保できるときだけ加える
（D は n≥7、E は n≥8。D=E=0 なら古典的 van Deemter に一致）。

H から理論段数 N、ピーク幅への関係（標準定義[文献8]）：

```
N = L / H
半値幅 W_h と N:   N = 5.54·(t_R / W_h)²   （5.54 = 8·ln2）
ベースライン幅 W_b = 4·t_R / √N            （分離度の式で使う）
```

> **統計メモ（なぜ幅が要るか）**：分離できているかは「ピーク間の距離」だけでなく
> 「各ピークの太さ」にもよる。離れていても太ければ重なる。だから幅もモデル化する。

### 3.7 分離度 Rs — クロスオーバー対応の定義

2つのピーク A・B の分離度（標準定義[文献8]）：

```
Rs = 2·|t_R(A) − t_R(B)| / (W_b(A) + W_b(B))
```

- 分子＝ピーク中心の距離、分母＝幅の和。Rs≥1.5 でほぼ完全分離、本プロジェクトは**安全側で 2.0** を要求。
- **絶対値 | |** を使うのがミソ。溶出順が入れ替わっても（クロスオーバーしても）正しく計算できる。
- 目的は TP を IP1・IP2 の**両方**から離すこと。だから
  `Rs1 = Rs(TP, IP1)`、`Rs2 = Rs(TP, IP2)` の**小さい方** `min{Rs1, Rs2}` を指標にする。
- ピーク同定は溶出順でなく**化合物の同一性**で固定する（クロスオーバーで順番が変わっても TP は TP）。

## 4. 使う統計の道具（統計検定2級レベルの解説）

### 4.1 回帰・最小二乗・計画行列

実測データから係数（a,b,c,…）を決めるのは**最小二乗法による回帰**。

- 各実験 i で「予測値」と「実測値」の差（残差）を考え、**残差の二乗和を最小**にする係数を選ぶ。
- 説明変数を並べた行列を**計画行列 X** と呼ぶ。ここでは保持モデルの説明変数は
  `[1, 1/T_K, φ, φ², φ/T_K]`。各行が1実験、各列が1つの説明変数。
- 係数のばらつき（不確かさ）は次式で表される[文献14]：

```
Var(係数) = σ²·(XᵀX)⁻¹     （σ² = 測定ノイズの分散）
```

> **統計メモ**：2級の「重回帰」。X の列が説明変数、最小二乗で係数を推定する。
> 上の式は「データの配り方（X）が良いほど係数が精密に決まる」ことを意味する（§5.4 D最適で使う）。

### 4.2 当てはまりの指標：R² と RMSE（SD との違い）

- **R²（決定係数）**：応答の変動のうちモデルで説明できた割合。1 に近いほど良い。
- **RMSE（二乗平均平方根誤差）**：予測が実測から平均的にどれだけずれるか。元の単位で読める。

```
RMSE = sqrt( 平均( (予測値 − 実測値)² ) )
```

- 例：t_R の RMSE が 0.5 秒 → 予測は平均的に実測から ±0.5 秒ずれる、という意味。

> **統計メモ（RMSE と SD の違い）**：同じ「二乗→平均→ルート」でも基準が違う。
> SD（標準偏差）は「実測値 − 実測値の平均」のばらつき（データ自体の散らばり）。
> RMSE は「予測値 − 実測値」のずれ（モデルの当たり外れ）。RMSE < SD なら
> 「平均で答えるよりモデルの方が当たる」＝モデルに価値がある。

### 4.3 多重共線性 —「係数は深読みしない」理由

温度範囲が狭い（例 40〜60℃）と、絶対温度 T_K では 313〜333 K で **1/T_K がほぼ一定**
（変化は約6%）。すると説明変数の `φ` と `φ/T_K` がほぼ比例し、見分けがつかなくなる。
これが**多重共線性**[文献14]。

- 影響：`c·φ` と `e·φ/T_K` の**個別の係数値が不安定**（実験のたびに大きく動く）。
- でも**合計 `c·φ + e·φ/T_K` は安定** → **予測値（t_R や Rs）は信頼できる**。
- 結論：本プロジェクトでは個別係数を「この成分は温度感受性が高い」などと深読みしない。
  品質は§4.2 の**予測スケールの RMSE** で評価する。

> **統計メモ（直感）**：「同じ音量を出すツマミが2つある」状態。どちらをどう回しても
> 最終音量（予測）は正しいが、各ツマミの目盛り（個別係数）は一意に決まらない。

### 4.4 自由度

- 1回の注入で3ピーク全部の t_R・W_h が測れる。だから自由度は「総本数」でなく**モデルごと**に数える。
- 保持（1ピーク）は5パラメータ（a,b,c,d,e）。Day1 の CCD 20本なら残差自由度は 20−5=15。
- 中心点6本は**純誤差**（同条件の繰り返しのばらつき）を測り、湾曲（2次の曲がり）の検定にも使う。

> **統計メモ（自由度）**：2級の「自由度＝データ数 − 推定するパラメータ数」。
> 残差自由度が十分あると、当てはまりの良し悪し（ラック・オブ・フィット）を検定できる。

## 5. 実験計画法（DoE）— どの条件で実験するか

### 5.1 なぜ計画的に振るか（応答曲面法 RSM）

闇雲に1因子ずつ変えるより、**複数因子を同時に系統的に**振る方が、少ない実験で
応答曲面（応答が条件でどう変わるか）と最適点・頑健領域を把握できる[文献10, 11]。

### 5.2 2次モデル用の計画（CCD と Box-Behnken）

3因子の代表的な2次モデル用計画として、本プロジェクトは次の3タイプから選べる[文献10, 11, 12, 13]：

```
中心複合計画(CCD): 頂点(factorial) 2³=8 ＋ 軸上点(axial) 6 ＋ 中心点 6 = 20 本（Day1）
Box-Behnken(BBD) : 各因子ペアの辺中点 12 ＋ 中心点 = 計画の角・軸点を使わない
```

符号化座標（中心=0、±1=指定範囲 low/high）→実値への変換は **頂点 ±1 が low/high に乗る標準規約**
（`real = center + 符号化値 × (range/2)`）。軸上点の距離 α で性質が変わる：

- **Box-Behnken**：角（頂点）も軸点も使わず、**全点が範囲内**に収まる。
  温度上限×高ACN のような「角の条件が物理的に無理」な系に向く[文献12]。
- **面心 CCD（α=1.0）**：頂点・軸点とも範囲の面上。範囲を絶対に超えたくないとき。
- **回転可能 CCD（α=2^{3/4}≈1.682）**：頂点が範囲の角、軸点はその外側へ伸びる。
  **回転可能性**（中心から等距離なら方向によらず予測分散が一定）を満たす古典的な良配置[文献11]。

保持モデルは実質「(T,φ) の2次曲面」なので、3因子 CCD=20本は2次モデル当てに教科書的サイズ[文献11]。

> **統計メモ（回転可能性）**：実験計画で「中心から同じ距離の点なら、どの向きでも予測の不確かさが
> 等しい」性質。軸点を α=2^{k/4}（k=因子数）に置くと得られる。範囲外に出る代償はあるが、
> 応答曲面のどの方向にも公平な精度が欲しいときに選ぶ。

### 5.3 ブロッキング（日間差の補正）

Day1（CCD）と Day2（追加実験）を別日にやると、移動相ロットや室温で**日間差**が混じる。
これをモデルに「日」を表す項 `+ δ·day` として入れ、Day2 冒頭に中心点3本を置いて
日間のズレ（オフセット）を測って橋渡しする[文献11]。

> **統計メモ（ブロック因子）**：2級の「ブロック化」。興味はないが結果に影響する要因（日）を
> ブロックとして取り込み、その変動を分離して本来の効果を正しく推定する。

### 5.4 D最適 augment（追加実験の選び方）

Day2 の追加点は **D最適計画**で選ぶ。D最適とは、§4.1 の `Var(係数)=σ²(XᵀX)⁻¹` を踏まえ
**det(XᵀX) を最大化**する点を選ぶこと（係数の信頼楕円の体積を最小化＝最も精密に決まる配置）[文献11]。

本プロジェクトでは目的に応じて2基準を用意：

- **(A) 係数精度のD最適**：保持モデルの実説明変数 `[1,1/T_K,φ,φ²,φ/T_K,1/u,u]` で det を最大化。
  このモデルは**係数について線形**なので、X は座標だけで決まり、**係数の値に依存しない**（頑健）。
- **(B) Rs境界標的の局所D最適**：最終目的の Rs は係数について**非線形**（§3.5 の指数のため）。
  Rs の係数感度（ヤコビアン）を Day1 のフィット値で評価し、Rs=2.0 の**境界近傍に点を集める**。
  デザインスペースの境界精度を直接上げる。

> **統計メモ（線形/非線形の見分け）**：モデルを各パラメータで偏微分して、結果にパラメータが
> 残れば非線形、残らなければ線形。`ln k = a+b/T_K+c·φ+…` は偏微分すると 1（aで）、1/T_K（bで）…と
> パラメータが消える＝**線形**。`t_R=(V_m/F)(1+e^{ln k})` は偏微分に e^{…}（＝パラメータ入り）が
> 残る＝**非線形**。線形だと「最適な実験配置が係数を知らなくても決まる」という利点がある。

## 6. デザインスペースと推奨条件（ICH Q8）

### 6.1 デザインスペースの定義

実測でフィットしたモデルを使い、T-φ-F 空間を細かい格子で評価し、各点が**合格条件**を
満たすか判定する：

```
合格 ⟺  min{Rs1, Rs2} ≥ 2.0   かつ   t_R(TP) ≤ 7.5 分
```

全合格点の集まりが**デザインスペース**[文献9]。
（「最遅ピーク ≤ 10分」はデータ取り段階の洗浄前制約であって、デザインスペースの合否には含めない。）

> **ICH Q8 の定義**[文献9]：デザインスペース＝「品質を保証できると実証された入力変数・工程
> パラメータの多次元的な組合せと相互作用の範囲」。承認後、その中での操作は変更とみなされない。

### 6.2 最大余裕点（推奨条件）

デザインスペースの中で、**境界から最も遠い点**を推奨条件にする（条件がブレても合格に留まる＝
最もロバスト）。「境界」は2種類：

1. 合格条件を破る不合格点（Rs不足・時間超過の崖）
2. 因子範囲の端（検証していない外側への外挿を避けるため、端も崖扱い）

各因子を範囲幅で 0〜1 に正規化してから距離を測り（単位差の吸収）、最寄りの崖までの距離が
最大の点を選ぶ。これは ICH Q8 の「ロバストな運転点」の考え方に沿う[文献9]。

**推奨条件の選び方は3基準から選べる**：

1. **最も頑健**（上記の最大余裕点）：不合格の崖から最も遠い点。
2. **t_R(TP) 最速**：各因子が運用中にブレうる幅 ±(δT, δφ, δF) を設定し、その**箱の全域が合格**な
   点の中で、目的ピークが最も速く出る条件を選ぶ（分析時間を短縮）。
3. **ACN 消費量が最小**：同じく箱全域が合格な点の中で、TP 溶出までの ACN 消費量
   `φ × F × t_R(TP)`（移動相通液量に ACN 分率を掛けた体積）が最小の条件を選ぶ（溶媒節約）。

2・3 の「箱の全域が合格」は、合格マスクを箱サイズで**収縮（erosion）**して判定する。
3D 図にはこの変動範囲の箱を破線で重ね、箱がデザインスペースの雲からはみ出していないこと
（＝設定したブレ幅すべてで合格）を目視できる。

### 6.3 外挿（参考）

因子範囲の外まで予測を広げるオプションもあるが、これは**実験で検証していない領域の予測**なので
信頼性が下がる。グラフでは検証範囲を枠で示し、推奨条件は必ず検証済みの元範囲内から選ぶ。

## 9. 用語集（統計検定2級補足）

- **回帰／最小二乗**：説明変数から応答を予測する式を、残差二乗和最小で当てる方法。
- **計画行列 X**：各行=1実験、各列=説明変数 を並べた行列。
- **R²**：応答の変動のうちモデルが説明した割合（0〜1）。
- **RMSE**：予測の平均的なずれ（元の単位）。SD はデータ自体の散らばり、RMSE はモデルの当たり外れ。
- **多重共線性**：説明変数どうしが強く相関し、個別係数が不安定になる現象。予測は保たれる。
- **自由度**：データ数 − 推定パラメータ数。残差自由度が当てはまり検定に必要。
- **交互作用**：ある要因の効果が別の要因の水準で変わること。
- **ブロック因子**：興味はないが影響する要因（例：実験日）を取り込み変動を分離する。
- **応答曲面法(RSM)／CCD**：複数因子を系統的に振り、応答曲面と最適・頑健領域を把握する計画。
- **D最適**：`det(XᵀX)` を最大化して係数を最も精密に決める実験配置。
- **デザインスペース**：品質が保証されると実証された条件の多次元範囲（ICH Q8）。
- **クロスオーバー**：条件により溶出順が入れ替わる現象。
