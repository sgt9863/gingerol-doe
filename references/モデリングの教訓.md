# モデリングの教訓 — 「理論的に影響する」≠「モデルに組み込むべき」

> このプロジェクトで得た、モデル構築のいちばん重要な教訓。
> **ある因子が理論的に応答へ影響するとしても、それをモデルに組み込むのが最適とは限らない。**
> 組み込むべきかは「理論的影響の有無」ではなく「**手元のデータがその項を支持するか**」で決める。

---

## 何が起きたか（幅モデルの例）

ピーク幅（段高 H）は、理論的には温度 T・溶媒比 φ・流速 F すべてに依存する
（van Deemter、C 項の保持係数 k 依存、水/ACN 粘度の φ 山型 など）。
そこで一度は幅モデルを `H = A + B/u + C·u + D·φ + E/T_K + G·φ²`（6パラメータ）まで拡張した。

しかし実データで検証すると：

- **N（理論段数）は条件でほぼ一定**（CV 5〜13%）。W_h は t_R にほぼ比例（r=0.96〜0.99）。
- **u（流速）は DoE の範囲が狭くほとんど動かない** → van Deemter の u 項は効果を検出できない。
- **AICc 後退法**で項を選ぶと、W_h の最良の説明変数は **t_R** で、u・φ・T の項は削除される
  （IP1 は「切片＋t_R」だけが最良）。
- **Q²（予測 R²・LOO）** で見ると、H 空間の拡張モデルは負〜低（過学習気味）、W_h を t_R で
  直接表すと高い。

→ 最終的に幅モデルは **`W_h = wc0 + wc1·t_R`（2パラメータ）** に簡素化した。
推奨条件（T・φ）はほぼ変わらず、デザインスペースの形もほぼ同じ。

---

## 一般化した教訓

1. **理論的影響 ≠ モデルに入れるべき項**。
   影響が「あるか」ではなく、その影響が **この実験条件・この範囲で観測可能なほど大きいか**、
   かつ **データ点数に対して推定に足るか** が判断基準。
   - 例：温度は段高に効くが、N がほぼ一定・F 範囲が狭い本条件では、幅に対して t_R 以上の情報を
     加えない。「効くはず」で入れると過剰パラメータになる。

2. **判断は客観指標で**：
   - **AICc**（小標本補正つき）で当てはめと複雑さのバランス。
   - **Q²（予測 R²・交差検証）**：R² だけ見ると過学習を見逃す。Q²≪R² は危険信号。
   - **lack-of-fit 検定**（中心点の純誤差と比較）でモデル形の妥当性。
   - **多重共線**（保持モデルの個別係数）は p 値・符号を不安定にする → 個別係数を深読みせず
     予測スケール（RMSE・Q²）で評価。

3. **モデルは「目的」に対して十分であればよい**。
   本プロジェクトの目的は Rs（分離度）と最適条件。Rs は t_R が支配的で、幅は t_R にほぼ従属。
   だから幅を精緻化しても目的への寄与は小さい。**目的関数への感度**を基準に労力を配分する。

4. **メカニズム（理論）の役割は "項の候補と関数形を与えること"**。
   どの候補を最終的に残すかはデータが決める。メカニズムモデルの強みは、少数データでも
   物理的に妥当な関数形（`ln k=a+b/T_K+…` 等）を与えられることで、全項を常に使うことではない。

---

## 具体的な結末（このプロジェクト）

| 項目 | 最終的な選択 | 理由 |
|---|---|---|
| 保持 `ln k` | `a + b/T_K + c·φ + d·φ² + e·φ/T_K`（フル） | AICc でほぼフルが支持。多重共線でも予測は安定 |
| 幅 `W_h` | `wc0 + wc1·t_R`（t_R のみ） | N ほぼ一定。AICc/Q² が t_R 単独を支持。u・φ・T 項は過剰 |
