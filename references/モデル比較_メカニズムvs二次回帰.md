# モデル比較 — メカニズムモデル vs 二次回帰（応答曲面）

> **目的**：本プロジェクトが採用する「クロマトグラフィーのメカニズムに基づくモデル」と、
> 一般的な「フル2次の応答曲面（二次回帰）」を、実データ（Day1 CCD 20本、10-gingerol）で
> 比較し、なぜメカニズムモデルを採用するのかを数値で裏づける。
>
> 結論を先に：**このデータ規模ではメカニズムモデルが明確に優位**（特に少数データと外挿）。
> in-sample の当てはまり（R²）では二次回帰も高く見えるが、交差検証（未知点予測）で差が出る。

---

## 1. 比較した2モデル

| | 説明変数 | パラメータ数（応答1つあたり） |
|---|---|---|
| **メカニズム（保持）** | `ln k = a + b/T_K + c·φ + d·φ² + e·φ/T_K` | 5 |
| **メカニズム（幅）** | `H = A + B/u + C·u + D·φ + E/T_K + G·φ²`（u=L·F/V_m） | 6 |
| **二次回帰（応答曲面）** | `y = β₀ + β₁T + β₂φ + β₃F + β₄T² + β₅φ² + β₆F² + β₇Tφ + β₈TF + β₉φF` | 10 |

- メカニズムは物理的に効く項だけを持つ（保持は温度×溶媒、幅は van Deemter＋補正）。
- 二次回帰は3因子のフル2次で固定的に10項。応答（t_R・W_h）を直接あてる。

---

## 2. 評価方法 — なぜ LOO-CV か

- **in-sample R²**（全データで当てて全データで評価）は、パラメータが多いモデルほど機械的に高くなる。
  二次回帰は10パラメータなので当てはまりは良く見えて当然で、これだけで比べると不公平。
- そこで **LOO-CV（Leave-One-Out 交差検証）**：1点を抜いて残りで学習し、抜いた点を予測。
  これを全点で繰り返した予測誤差（RMSE）が「**未知の条件での予測力**」を表す。
  過学習していると、in-sample は良いのに LOO-RMSE が跳ね上がる。

> **統計メモ**：LOO-CV は「手持ちデータの内側での擬似的な予測テスト」。真の外挿（範囲外）とは別物だが、
> 少数データでの過学習の有無をあぶり出すのに有効。範囲外の本当の外挿性能は、別途 範囲外の実測が要る（§5）。

---

## 3. 結果

### 3.1 保持時間 t_R

| ピーク | メカ(5param) R² / LOO-RMSE | 二次(10param) R² / LOO-RMSE |
|---|---|---|
| TP (n=18) | 1.000 / **2.5秒** | 1.000 / 4.5秒 |
| IP1 (n=13) | 1.000 / **2.9秒** | 1.000 / **156秒** 💥 |
| IP2 (n=18) | 1.000 / **3.2秒** | 1.000 / 4.4秒 |

### 3.2 ピーク幅 W_h

| ピーク | メカ(6param) R²(W_h) / LOO-RMSE | 二次(10param) R²(W_h) / LOO-RMSE |
|---|---|---|
| TP (n=18) | 0.936 / **0.66秒** | 0.981 / 0.90秒 |
| IP1 (n=13) | 0.973 / **0.58秒** | 0.995 / **2.92秒** 💥 |
| IP2 (n=18) | 0.999 / **0.16秒** | 0.998 / 0.40秒 |

## 4. 読み取り

- **in-sample R² は二次回帰の方が高い**（幅で顕著）。パラメータが多いので当然。
- **しかし LOO（未知点予測）はメカニズムが全勝**。とくに **IP1（13点）で二次回帰が破綻**
  （t_R の LOO が 156秒＝約2.6分）。10パラメータを13点に当てるとほぼ飽和し、1点抜くと予測が暴れる
  ＝典型的な**過学習**。
- メカニズムは 5〜6 パラメータで、少ない点でも安定して外れない。
- **t_R はどちらも in-sample R²=1.000**（保持が滑らかなため）。in-sample だけ見ると二次回帰でも十分に
  見えるのが落とし穴で、LOO で初めて差が出る。

### メカニズムが有利な理由（3点）

1. **パラメータが少ない**：物理的に効く項だけ（保持5・幅6）。少数データ・欠損に強い。
   （今回は欠損で IP1 が13点まで減ったのが二次回帰に厳しかった。）
2. **外挿が物理的に妥当**：二次回帰は範囲外で放物線が反り返り、t_R が非単調・負になりうる。
   メカは `ln k` 直線＋指数で単調・正値が保証される。デザインスペースの外挿オプションと相性が良い。
3. **係数に意味がある**：「温度の効き b」「溶媒の効き c」等として解釈・転用できる
   （別化合物へ config 差し替えで移せる）。二次回帰の交差項は物理的解釈が薄い。

---

## 5. 限界と今後

- 本比較は **LOO-CV（データ内側の擬似予測）** であり、範囲外の**真の外挿性能**ではない。
- 真の外挿を評価するには、因子範囲の外（例：T=40/60℃、φ=0.38/0.52 など）で**実測を数点追加**し、
  内側データで学習したモデルで予測 → 外側実測と比較する必要がある（データ取得待ち）。
- 十分な点数（3因子で欠損なく20点以上）を確保できれば、二次回帰でも実用になりうる。
  本データは欠損により少数点となり、メカニズムの優位が際立った。
