# モデル比較 — Eq27（Neue–Kuss＋温度） vs 本プロジェクトのメカニズムモデル

> **目的**：den Uijl ら (J. Sep. Sci. 2020, 44, 88–114) の総説で挙げられる 2次元(φ,T)保持式
> **Eq 27（Neue–Kuss ＋ 温度依存, 4パラメータ, 非線形）**を実装し、本プロジェクトの保持モデル
> （5パラメータ, 線形）と**実データ（Day1 CCD 20本）で多角的に比較**する。
>
> ※ 幅モデルは両者に共通の `W_h = wc0 + wc1·t_R` を使い、保持式だけの差を評価する。

---

## 1. 比較した2式

**Eq 27（Neue–Kuss ＋ 温度）— 4パラメータ・パラメータ非線形**
```
ln k = ln k0 + D/T_K + 2 ln(1 + a·φ) − (a + D/T_K)·S·φ / (1 + a·φ)
   params: ln k0, D, a, S     （a が分母 → 非線形。多初期値の非線形最小二乗が必要）
```

**本プロジェクトのメカニズム — 5パラメータ・パラメータ線形**
```
ln k = A0 + b/T_K + c·φ + d·φ² + e·φ/T_K     （OLS一発。φ² で曲率を近似）
```

Eq27 は φ の曲がりを Neue–Kuss の飽和形で、当モデルは多項式 φ² で表す。温度はどちらも 1/T 形。

---

## 2. 結果（実データ・現行の幅モデル共通）

| 観点 | Eq27 (4p, 非線形) | メカニズム (5p, 線形) | 優 |
|---|---|---|---|
| in-sample R²(t_R) TP/IP1/IP2 | 0.9998 / 0.9997 / 0.9997 | 同等 | 引分 |
| in-sample RMSE(t_R) [秒] | 1.20 / 1.23 / 1.70 | **1.15 / 1.12 / 1.64** | わずかにメカ |
| **AICc**（lnkスケール・省パラ考慮） | **−181.9 / −123.2 / −175.0** | −179.0 / −120.8 / −171.3 | **Eq27** |
| **LOO-RMSE(t_R) [秒]** | **1.73 / 1.94 / 2.39** | 2.45 / 2.91 / 3.17 | **Eq27** |
| Q²(ln k, LOO) | 0.999 / 0.999 / 0.999 | 0.999 / 0.996 / 0.999 | ほぼ引分 |
| **Rs_min の LOO**（全ピーク揃う12点）RMSE/MAE/max | 0.149 / 0.126 / 0.340 | **0.109 / 0.097 / 0.210** | **メカ** |
| フィットの頑健さ | 非線形・多初期値要（今回はRSSばらつき~3e-4で安定） | **OLS一発・係数非依存** | メカ |

（実測 Rs_min 範囲 1.63–2.17。Eq27 推定値の例 TP: lnk0=−1.23, D=6217, a=3.19, S=4.14）

---

## 3. 読み取り

1. **単一ピークの保持予測は Eq27 が上**。1パラメータ少ない（4 vs 5）のに、AICc も LOO-RMSE も全ピークで
   Eq27 が勝つ。Neue–Kuss の飽和形 `2ln(1+aφ)` が、多項式 `φ²` より保持カーブを素直に外挿できるため。
   **理論的にも実測でも、"保持そのもの" では Eq27 が優れた式**と正直に認められる。

2. **しかし目的関数 Rs では逆転してメカニズムが勝つ**（LOO 0.109 vs 0.149）。Rs は**ピーク間の差**なので、
   3ピークを**同じ線形基底**で当てるメカニズムは誤差が相関し、差を取ると打ち消し合う。Eq27 は各ピークを
   独立に非線形フィットするため絶対保持は当たっても差（Rs）に誤差が残る。
   → **「絶対保持が上手い ≠ 分離度が上手い」**。目的関数への感度で評価すべき、という本プロジェクトの
   一貫した教訓（`モデリングの教訓.md`）と同じ構図。

3. **運用コストはメカニズムが圧勝**。Eq27 は `a` が分母に入る非線形式で、フィット・LOO・D最適の各所で
   多初期値の非線形最適化が要る（今回は収束安定だったが機械が重い）。当モデルは OLS 一発、D最適計画も
   係数値に依存しない。少数データ・自動化・別化合物への転用では明確に有利。

---

## 4. 結論

- **保持の精度・パラメータ効率**を最優先するなら **Eq27（NK＋温度）が一枚上**。広い φ 域・強い外挿では
  差が開く可能性が高く、"保持予測の base 式" を選ぶなら Eq27 は有力候補。
- **本プロジェクトの目的（Rs・デザインスペース・少数データ・OLSの頑健さ・自動化・再利用）**では、
  **メカニズムモデルが実利的に最適**という結論は揺らがない（Rs予測でむしろ勝ち、実装が桁違いに簡単）。
- 一言でいえば **Eq27＝「保持の精度」で勝ち、当モデル＝「目的（分離度）と運用性」で勝ち**。
  用途が「単一化合物の保持を精密予測」なら Eq27、「分離条件の最適化を頑健・自動で」なら当モデル。

---

## 5. 限界

- LOO-CV はデータ内側の擬似予測で、範囲外の真の外挿ではない（`モデル比較_メカニズムvs二次回帰.md §5` と同じ注意）。
  Neue–Kuss の外挿優位は理論的には期待できるが、本データ（狭い φ 域 0.42–0.50）では検証しきれない。
- n=13–18 と小さく、LOO-tR の差（~0.5–0.7秒）や Rs の差（~0.04）は小さい。実用差の判断は用途次第。
- Eq27 は本データでは多初期値で安定収束したが、データや初期値次第で局所解に嵌るリスクは線形式より高い。
