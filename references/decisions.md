## プロジェクト概要

- **目的**：漢方中の 10-gingerol を HPLC で頑健に分離・定量するため、カラム温度 T・流速 F・溶媒比 φ（水:ACN）の3条件を統計的に最適化し、デザインスペースを3Dで可視化する
- **現行条件**：T=75℃ / F=0.6 mL/min / 水:ACN=60:40（φ=40% ACN）アイソクラティック
- **問題**：目的ピーク(TP)が前後の夾雑ピーク(IP1, IP2)に挟まれ、条件ブレで分離度が落ちるリスク。クロスオーバーあり。

---

## ピークの命名

| 略称 | 意味 | 現行条件での溶出順 |
|------|------|------------------|
| IP1 | Interfering Peak 1（夾雑ピーク1） | 1番目 |
| TP | Target Peak = 10-gingerol（目的ピーク） | 2番目 |
| IP2 | Interfering Peak 2（夾雑ピーク2） | 3番目 |

## 確定した設計判断

### 指摘1：流速 F のモデル上の扱い（確定）

- k（保持係数）は T と φ のみで決まる。F は保持に影響しない
- 保持モデル：`ln k = a + b/T[K] + c·φ`（必要なら +d·φ²） × 3本（IP1, TP, IP2）
- 保持時間の復元：`t_R = (V_m / F) × (1 + k)`
- 幅モデル：`W(T, φ, F)` × 3本（F が主役。理論段数 N を介す案が安定）
- V_m はカラム固有定数。不保持物質（ウラシル等）を現行条件で1回注入して確定

### 指摘2：デザインスペースの定義（確定）

- **合格基準**：`min{Rs1, Rs2} ≥ 2.0`
- **最適条件の選び方**：デザインスペース（合格領域）の中で、**境界からの最短距離が最大になる点**（最大余裕点）を採用
- 根拠：ICH Q8。条件ブレに対して最もロバストな点を選ぶ

### 指摘3：Rs1/Rs2 のクロスオーバー対応（確定）

- Rs は溶出順でなく**化合物の同一性**で定義する
- `Rs1 = 2·|t_R(TP) − t_R(IP1)| / (W_TP + W_IP1)`
- `Rs2 = 2·|t_R(TP) − t_R(IP2)| / (W_TP + W_IP2)`
- 絶対値 |Δt| を使うため、溶出順が入れ替わっても Rs は常に正しく計算される

---

### 指摘4：探索範囲と合格条件（確定）

**カラム情報**
- Waters ACQUITY UPLC BEH C8, 130Å, 1.7µm, 2.1mm × 100mm
- 耐久上限 90℃ / 実用推奨上限 80℃（ACN 沸点 82℃ への安全マージン）

**探索範囲（暫定）**

| 因子 | 下限 | 中水準 | 上限 |
|------|------|--------|------|
| T（温度） | 40℃ | 50℃ | 60℃ |
| φ（ACN 比率） | 38% | 45% | 52% |
| F（流速） | 0.4 mL/min | 0.6 mL/min | 0.8 mL/min |

- F の水準範囲は `t_R = V_m(1+k)/F` の反比例から計算。V_m・k（中水準）の実測値が出たら `F_low = V_m(1+k)/tR_high`、`F_high = V_m(1+k)/tR_low` で更新する
- 各因子の目標効果量：t_R ± 1分（端から端まで変えたときの変化量）
- 現行条件（75℃ / 40% ACN）より低温・高 ACN 側にシフト

**合格条件（デザインスペース判定に使う）**

| 条件 | 閾値 | 備考 |
|------|------|------|
| Rs = min{Rs1, Rs2} | ≥ 2.0 | |
| t_R(TP) | ≤ 7.5分 | ばらつき込みの分析時間上限 |
| t_R(IP2) | ≤ 10.0分 | 洗浄ステップ開始時間 |

**水準範囲の最終確定フロー**
1. 指摘7の実測後にモデルをフィット
2. 各合格条件の等高線を T-φ-F 空間に描く
3. 全条件を満たす領域をデザインスペースとして確定
4. 感度分析で各因子の t_R への効果量を確認し、水準範囲を微調整して D最適追加実験へ
