# HPLC デザインスペース最適化スキル

## このスキルがすること
HPLC の3条件（T=カラム温度・φ=移動相 ACN 比率・F=流速）を最適化し、
**目的ピーク(TP)を前後の夾雑ピーク(IP)から Rs≥2.0 で頑健に分離する条件領域（デザインスペース）**を
求めて 3D 可視化する。実験計画（CCD→D最適）の生成、実測データのフィット、最適化、可視化まで一気通貫。

## いつ使うか
- 夾雑ピークに挟まれた目的成分の HPLC 分離条件を、実験計画法で頑健に最適化したい
- 保持時間 t_R・分離度 Rs を T/φ/F の関数として予測するメカニズムモデルを組みたい
- ICH Q8 のデザインスペースと、その中で最もブレに強い運転点（最大余裕点）を求めたい
- ピークのクロスオーバー（溶出順の入れ替わり）がある系を扱いたい

## 使い方（2通りの入口・中核ロジックは共通）
中核ロジックは numpy/scipy/statsmodels/pandas のみ。同じコードを3経路で使える。

1. **Streamlit Web アプリ**（推奨・対話的）
   ```
   pip install -r requirements.txt
   streamlit run app.py
   ```
   タブ式の正規フロー：①CCD計画→②Day1フィット→③D最適augment→④最終解析（＋デモ）。
   各段で実験データ入力用の Excel 雛形をダウンロードできる。

2. **Python in Excel / Jupyter セル**
   `scripts/01_model.py`〜`04_optimize.py` の関数をセルに貼り付けて使う
   （自作モジュール import 不可・ファイル出力不可の制約に対応するため、中核は plotly 非依存）。
   対話的3D・ファイルDLが要る場面は Streamlit 側で担保。

## 別化合物への転用（ここだけ変える）
**`config.example.yaml` を `config.yaml` にコピーして値を書き換えるだけ**。固定ロジック（scripts/）は触らない。
変える項目：
- `peaks` … 目的ピーク名・夾雑ピーク名（数は可変）
- `factors` … T/φ/F の探索範囲（low/center/high）
- `column` … カラム寸法（内径・長さ）・空隙率（V_m を幾何推算）・幅定義
- `acceptance_criteria` … Rs_min 下限・t_R(TP) 上限（`tR_last_max` は既定 null＝デザインスペース合否に含めない）
- `design` … 中心点数・D最適追加点数・橋渡し中心点・1日本数上限

## 数理モデルの要点
- 保持（係数について線形）：`ln k = a + b/T_K + c·φ + d·φ² + e·φ/T_K (+ δ·day)`、各ピーク3本
  - `e·φ/T_K` は温度×溶媒の交互作用＝クロスオーバーの源
- 保持時間：`t_R = (V_m/F)(1+k)`、ボイド時間 `t_0 = V_m/F`
- 幅（van Deemter）：`H = A + B/u + C·u`（u=線速度）、`N=5.54·(t_R/W_h)²`、`W_b=4·t_R/√N`
- 分離度（クロスオーバー対応・化合物同一性）：`Rs = 2·|t_R(A)−t_R(B)|/(W_b(A)+W_b(B))`、目標 `min{Rs1,Rs2}`
- デザインスペース：`Rs_min≥2.0` かつ `t_R(TP)≤上限`。境界＋因子範囲の端を崖として、最遠点＝最大余裕点（ICH Q8）

## 実験計画（DoE）
- Day1：CCD（頂点8＋軸6＋中心点）。Day2：D最適 augment＋橋渡し中心点（別日のブロッキング `+δ·day`）
- D最適は2基準を選択可：
  - (A) メカニズムモデル基準（係数精度・線形なので係数値に非依存・頑健）
  - (B) Rs境界標的の局所D最適（Rsヤコビアンで境界近傍に集中・Day1フィット要）
- 保持は「(T,φ)の2次曲面」なので3因子CCD=20本が2次モデル当てに教科書的サイズ

## スクリプト構成（固定ロジック）
| ファイル | 役割 | 依存 |
|---|---|---|
| `scripts/01_model.py` | 数理モデル（k, W, Rs） | numpy |
| `scripts/02_design.py` | CCD＋ブロッキング＋D最適(A/B) | numpy/pandas |
| `scripts/03_fit.py` | フィット（保持3・幅3・day推定） | numpy/pandas/statsmodels |
| `scripts/04_optimize.py` | デザインスペース判定＋最大余裕点＋外挿 | numpy/scipy |
| `scripts/05_designspace.py` | plotly 3D（雲＋等高線・Viridis） | plotly/matplotlib |
| `app.py` | Streamlit 入口（01〜05 を呼ぶだけ） | streamlit/plotly |

## 注意・設計上の約束
- 中核ロジック（01〜04）は numpy/scipy/statsmodels/pandas のみ。plotly/matplotlib は 05 と app.py 専用
- 保持の個別係数は実験範囲が狭いと多重共線。係数の深読みは禁物、品質は予測スケールの RMSE で見る
- 外挿（因子範囲外の予測）は可能だが検証外。推奨条件は必ず検証済みの元範囲内から選ぶ
- 配色は Viridis。デザインスペース内は色を引き伸ばし（紫=ギリギリ→黄=余裕大）、失格域は無色透明
