HPLC 保持モデル · 理論ノート

アイソクラティック保持回帰モデルの構築

保持係数 k から出発し、ファントホッフ（温度）× LSS（溶媒比）を積み上げて、 温度・溶媒比の多因子回帰式へ。等溶媒（グラジエントなし）条件でどう予測式を組み立てるかを、 導出の順に追う。

§0

「等溶媒」とは — なぜ回帰で解けるのか

アイソクラティック（等溶媒・isocratic）とは、分析のあいだ移動相の組成 （有機溶媒比 φ）をずっと一定に保つ運転方式のこと。 時間とともに φ を変えるグラジエントの対義語である。 等溶媒では 1 回の分析は温度 T・溶媒比 φ・流速 F が 固定された空間内の 1 点に対応する。

だから「いくつかの等溶媒条件で保持時間を測り、保持の強さを条件 (T, φ, F) の関数として当てはめる」――これはそのまま 重回帰の問題になる。以下では、その回帰式を勘でなく クロマトグラフィーの物理化学から組み立てる。

なぜ物理式か 汎用の多項式 y = a + bT + cφ + … を当てることもできる。だが物理式を使うと (1) 少ない実験でも外挿・条件間の挙動が妥当になりやすく、(2) 係数が「温度の効き」「溶媒の効き」 として意味を持ち、(3) 溶出順の入れ替わり（クロスオーバー）を交互作用として自然に表せる。

§1

保持係数 k と保持時間 tR

保持係数 k は、成分が固定相にどれだけ強く引き留められるかを表す無次元量 （大きいほど遅く出る）。保持されない物質が通り抜けるボイド時間 t0 と、ピークの出る保持時間 tR の関係は：

tR = t0(1 + k),    t0 = VmF (1)

ここで Vm はカラム内で移動相が占める体積（死容量）、F は流速。 逆に解けば、実測 tR から k が求まる： k = t_R·F/V_m − 1。ここが土台で、以降は 「k が T と φ でどう決まるか」を積み上げる。

効くのは T と φ k は温度と溶媒比で決まり、流速 F は k を変えない（F は t0 を通じて時間軸を伸縮させるだけ）。F はおもにピーク幅の側で効く（§6）。

§2

温度の効き — ファントホッフ式

分配平衡の熱力学（ファントホッフの関係「平衡定数の対数は 1/T に比例」）を クロマトに当てると、保持の対数は絶対温度の逆数に比例する：

ln k = a + bTK (2)

TK = 温度[℃] + 273.15（ケルビン）、a は切片、b は温度項の係数。 逆相では溶質移行が発熱的なので b > 0 となり、低温ほど保持が強い。 実際は曲がる場合もあるが、疎水性相互作用が支配的なら直線で近似でき、必要なら高次で補正する。

統計メモ · 変数変換 「ln k を 1/TK に対して直線で表す」は、統計2級の単回帰と同じ発想。 縦軸 ln k・横軸 1/TK をとると直線になる、という変数変換である。

§3

溶媒の効き — LSS（線形溶媒強度）

逆相 HPLC では、有機溶媒（ACN）の体積分率 φ を増やすほど成分は速く出る。 広い範囲で ln k が φ の一次式になることが知られる （Linear Solvent Strength, Snyder）：

ln k = ln kw − S·φ (3)

ln kw は φ=0（純水）での保持、S は溶媒強度の傾き （大きいほど φ に敏感）。これを §2 の温度項に足し合わせる。 係数 c が −S に相当し、曲がりの補正に 2 次項 d·φ² を足せる：

ln k = a + bTK + c·φ ( + d·φ2 ) (4)

§4

温度 × 溶媒 とクロスオーバー

温度と溶媒の効きは独立でない――溶媒の傾き S が温度で変わる。 これを交互作用項 e·(φ/TK) で表す：

ln k = a + bTK + c·φ + d·φ2 + e·φTK (5)

この e·φ/T_K がピークごとに違うと、ある条件では IP1 < TP < IP2、別の条件では順番が入れ替わる――クロスオーバーが生じる。 分離度の定義（§7）はこれに耐える形にする。

統計メモ · 交互作用 ある要因の効果が別の要因の水準で変わること。ここでは「温度の水準によって、溶媒を増やしたときの 保持低下の度合いが変わる」という意味。

§5

なぜ最小二乗で解けるのか — パラメータについて線形

式 (5) は φ や 1/TK について見れば曲線だが、 推定したい係数 a,b,c,d,e については「足し算」＝線形である。 だから応答を y = ln k、説明変数を [1, 1/T_K, φ, φ², φ/T_K] としたふつうの重回帰（最小二乗）で係数が決まる。

各行を 1 実験、各列を 1 説明変数として並べた計画行列 X を作り、残差二乗和を 最小化する。係数の不確かさは次式で表される：

Var(係数) = σ2·(X⊤X)−1 (6)

σ2 は測定ノイズの分散。この式は「データの配り方（X）が良いほど係数が精密に決まる」 ことを意味し、実験計画（CCD・D 最適）や Rs の信頼区間の土台になる。

統計メモ · 多重共線性 温度範囲が狭いと TK は 313–333 K で 1/TK がほぼ一定（変化 ~6%）。 すると φ と φ/T_K がほぼ比例し、 個別の係数値は不安定になる。ただし和は安定なので予測（tR・Rs）は信頼できる。 だから個別係数は深読みせず、予測スケールの RMSE / Q² で品質を見る。

§6

保持時間への復元と「非線形」の正体

フィットした ln k から k を戻し（k = e^{ln k}）、 式 (1) で保持時間に直す：

tR = VmF(1 + e ln k) (7)

ここで指数 e^{ } が入るため、tR は係数について 非線形になる。重要な使い分けはこうだ――フィット自体は ln k が線形 なので単純な線形回帰でよい。非線形になるのは、係数から tR や Rs を 予測する出口の段階だけ（Rs の信頼区間や境界標的の D 最適で効いてくる）。

§7

ループを閉じる — 幅と分離度

分離できているかは「ピーク間の距離」だけでなく「各ピークの太さ」にもよる。本モデルは、理論段数 N が条件でほぼ一定という実データ事実（相関 r = 0.96–0.99）を踏まえ、半値幅 Wh を保持時間に比例させる：

Wh = wc0 + wc1·tR (8)

ベースライン幅 Wb に直し、目的ピーク TP と各夾雑 IP の分離度 Rs を 絶対値付きで定義する（溶出順が入れ替わっても正しい）。目標は全 IP に対する Rs の 最小値を 2.0 以上にすること：

Rs = 2·|tRTP − tRIP|WbTP + WbIP,   目標 minIP Rs ≥ 2.0 (9)
クロスオーバー対応 絶対値 | | がミソ。φ や T を変えて IP1 が TP を追い抜いても、十分に離れていれば Rs は大きいまま評価され、その条件も合格・最適点候補として拾える。ピーク同定は溶出順でなく 化合物の同一性で固定する（順番が変わっても TP は TP）。
